# Dino-Game

- A Python version of the Chrome Dinosaur Game using Pygame.  
- Dinosaur can run, jump, and duck to avoid cacti and birds.  
- Score increases as the game speed gets faster.  
- When hitting an obstacle, a quiz question appears.  
- Correct answers allow the game to continue, wrong answers end it.  
- Includes start screen, gameplay, and final score screen.
